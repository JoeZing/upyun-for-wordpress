Auto save the attachments to Upyun Cloud Storage

== Description ==

1，本插件只针对又拍云储存
2，本插件在3.5版本的基础上开发

== Installation ==

1，申请又拍云：http://www.upyun.com
2，创建一个空间
3，根据插件选项的说明，填写又拍云后台中的相关参数
4，上传本地的附件并将URL转换为又拍云的URL
